
To install the ordpatt2 package without testing functions, run the following code:

remotes::install_github("https://github.com/isgauper/ordpatt2",build_vignettes = TRUE)

You may run into issues if there are missing package dependencies, such as  "There is no package called 'ggthemes'" 
If so try searching in the error output and installing the missing package.


For the testing plan and necessary code, see the 'Testing Plan - ordpatt2.pdf' in the main 
portion of the GitHub repository: https://github.com/isgauper/ordpatt2/blob/main/Testing%20Plan%20-%20ordpatt2.pdf

If you run into any issues when installing or testing, please email me at isgauper@gmail.com.
